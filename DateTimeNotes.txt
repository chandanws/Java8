Java8 Date time related examples

Every Java developers should know about new date and time related APIs introduced in Java 8
Following classed are important from developers perspective
All classes in java.time package are immutable and thread-safe 

1) Instant
2) LocalDate
3) LocalTime
4) LocalDateTime
5) ZonedDateTime

Few helper classes

1) MonthDay
2) YearMonth
3) Clock
4) Period
5) DateTimeFormatter
